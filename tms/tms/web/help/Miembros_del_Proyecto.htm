Miembros del Proyecto

También en la pantalla de Inicio puede ver los miembros que integran la compañía  y al mismo tiempo poder cambiar los permisos, la contraseña,  o ver como se encuentra la carga de trabajo para cada uno de ellos o también ver sus horarios de trabajo.

: Si el check de selección se encuentra marcado indica que va a trabajar con ese miembro del equipo.

Nombre: Nombre completo del miembro del equipo de trabajo del proyecto.

Usuario: Nombre con el cual el sistema reconoce al usuario para dejarlo ingresar al T.M.S.

Correo Electrónico: Nombre del correo electrónico donde el miembro recibe sus correos.

Perfil: Perfil que tiene el miembro en el T.M.S. Puede ser ADMIN, Gerente de Proyecto, Usuario del cliente, Usuario de la empresa.

Horarios: Mediante esta opción Usted puede ver los días y horas de trabajo que tiene el miembro del equipo asignado para ese proyecto.

Horarios El sistema muestra la pantalla con las fechas de trabajo asignadas a los miembros del equipo.

Posiciónese en el campo fecha que desea consultar como se señala en la pantalla anterior por una flecha para ver el detalle de las tareas como se muestra en la siguiente pantalla:

Agenda: Mediante esta opción Usted puede ver la agenda de trabajo para el proyecto.

El sistema muestra la pantalla con las fechas de trabajo asignadas, también puede ver las tareas, el detalle del proyecto o enviar un mensaje al dueño del proyecto (campos señalados por las flechas)..

Posiciónese en los campos Hora o Tema para ver el detalle del evento como se muestra en la siguiente pantalla:

Posiciónese en los campos Tarea para ver el detalle del evento.

Posiciónese en el campo Proyecto para ver la información detallada del proyecto .

Posiciónese en el campo Dueño para enviar un correo en Outlook al encargado directo del proyecto.

 RENDIMIENTO

Usted puede observar el rendimiento del miembro del equipo de trabajo, simplemente indicando un rango de fecha inicial y final para que el sistema ejecute un proceso interno.

Fecha inicio: Fecha de inicio a partir de la cual.

Fecha final: Fecha hasta la cual desea consultar el rendimiento.

Haga click para visualizar el calendario.

El sistema genera la siguiente pantalla con el tiempo estimado, tiempo real y la variación absoluta del rendimiento del miembro del equipo..

 CARGA

Usted puede observar la carga de trabajo que tiene asignada el miembro del equipo de trabajo, simplemente indique un rango de fecha inicial y final y el sistema ejecuta un proceso interno.

Fecha inicio: Fecha de inicio a partir de la cual.

Fecha final: Fecha hasta la cual desea consultar el rendimiento.

Haga click para visualizar el calendario.

El sistema genera la siguiente pantalla con las tareas asignadas, fecha inicio, fecha final, tiempo estimado y el tiempo agendado.
