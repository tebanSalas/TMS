Mis Clientes

Esta pantalla le presenta la lista de todos los clientes que se encuentran en la base de datos La información inicial mostrada es Nomrbe del cliente, Teléfono y Web Site.

Ver los detalles de la organización de un cliente

Posicione el cursor sobre el nombre del cliente que desee consultar haga click sobre el mismo y el sistema le abrirá la siguiente pantalla, con la información detallada de la emprea, además los proyectos y los usuarios asignados para ese cliente.

Proyectos del Cliente

En la parte inferior de la pantalla podrá ver los proyectos que tiene asignado el cliente, con su correspondiente Estado, Dueño del Proyecto y Web Site.

Si lo que desea es agregar un proyecto para el cliente refiérase a Creando un Proyecto.
