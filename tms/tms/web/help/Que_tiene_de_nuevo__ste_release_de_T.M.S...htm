¿QUE TIENE DE NUEVO ESTE RELEASE CRM/SOS ?

CRM/SOS es un producto en constante evolución, dentro de las principales características de esta versión tenemos las siguientes.

Manejo de los ciclos administrativos

Manejo de los ciclos administrativos posterior al cierre de la venta, logrando de esta manera sincronizar la información en toda la organización y minimizar el tiempo de respuesta a los clientes internos y-o externos.

Creación y control de las campañas de mercadeo       

CRM/SOS permite la creación y control de las campañas de mercadeo, dentro de sus principales ventajas tenemos.

Asignación de recursos

Control de costos directos e indirectos

Opción de mailmerge para enviar correo masivo a los contactos, sean archivos

Precio más cómodo  

Ahora CRM/SOS puede ser adquirido en forma EMPAQUETADA con la base de datos Oracle Standard Edition One a una fracción del costo original

Envío de mensajes  

Esta versión permite enviar mensajes via SMS en aquellos países que cuenten con el servicio.

Recuerde:  Si se registra como usuario de CRM/SOS tendrá siempre la información actualizada de los últimos adelantos del producto y acceso a nuestro centro de servicio mundial.
