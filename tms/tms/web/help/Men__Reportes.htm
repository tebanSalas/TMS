Generando un Reporte

En esta pantalla se van a definir los diferentes filtros para la generación de un reporte de los proyectos, las personas a las que se asignaron los mismos, los estados en los que se encuentran, adicionalmente puede generar los reportes también seleccionando la prioridad de los mismos. Estos reportes son muy útiles para el gerente de proyectos sobre todo para saber cuáles proyectos se encuentran atrasados, cuáles tareas ya fueron finalizadas y saber en que áreas debe dar apoyo para lograr cerrar el proyecto.

Descripción de campos

Proyecto: Seleccione el proyecto sobre la que se va a generar el reporte.

Asignado a: Seleccione la persona que tiene a su cargo el proyecto.

Fecha Inicio: El sistema permite definir un rango de fechas que es el que se toma en cuenta para la generación del reporte.

Todas las fechas:Toma en cuenta todas las fechas que tiene esa cuenta y ese proyecto.

Entre las fechas: Seleccione el rango de fechas para la generación del reporte

Haga clic y el sistema le muestra el calendario para agendar

Estado: Es el posible estado en que se encuentra un proyecto.

Selecionar Todo:Genra en el reporte todos lo proyectos que se encuentran en la base de datos no importando su estado.

Finalizado por Cliente: Genera únicamente los proyectos que han sido finalizados por el cliente.

Finalizado: Genera los proyectos que se encuentran finalizados.

No Iniciado: Toma en cuenta los proyectos que tengan el estado de No Iniciado.

Inicado: Genera todos los estados de iniciado.

Suspendido: Toma en cuenta todos los que se encuentren con estado de suspendido.

Cotización: Genera solamente las cotizaciones.

Cotización enviada: Genera las cotizaciones que fueron enviadas.

Cotización rechazada: Genera las cotizaciones que fueron rechazadas.

Cotización aceptada: Genera solamente aquellas cotizaciones que fueron aprobadas.

Cobrar: Toma en cuenta todas las cotizaciones que fueron aprobadas y deben de ser cobradas.

Cobro enviado: Gernera las gestiones de cobro que ya fueron enviadas.

Procesando cobro: Toma en cuenta todas los proyectos a los que se les está generando el cobro para ser enviado.

Propagar Cambio: Este campo se utiliza para que una vez que el cambio en el sistema se haya terminado se propague para todos os clientes.

SI: Propaga el cambio a todos los clientes.

NO: El cambio se hace pero no se propaga en todos los clientes.

--: El cambio se está haciendo pero no se ha terminado

Fecha Final:   El sistema permite definir un rango de fechas final que es el que se toma en cuenta para la generación del reporte.

Todas las fechas:Toma en cuenta todas las fechas que tiene esa cuenta y ese proyecto

Entre las fechas: Seleccione el rango de fechas para la generación del reporte

Haga clic y el sistema le muestra el calendario para agendar

Prioridad: Como los proyectos tienen asignado un status de prioridad, mediante este campo se puede seleccionar la prioridad de los reportes que desea generar.

Genera el reporte con los filtros indicados anteriormente.
