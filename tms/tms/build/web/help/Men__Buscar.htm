Esta sección se utiliza para realizar criterios de búsqueda de acuerdo a los operadores.  El botón de buscar en TMS permite realizar las siguientes operaciones.

Una consulta se divide en dos tipos: términos y operadores. Los términos se subdividen en dos tipos: “términos sencillos” y “frases”

Un término es una palabra simple, como por ejemplo: “hola” o “proyecto” El usuario debe indicar la palabra que desea buscar en el sistema y colocar el check de selección de las áreas donde desea realizar la búsqueda

Una frase es un grupo de palabras rodeadas por comillas dobles como “proyecto uno".
