Navegando en el Producto

Lo único que debe tener en cuenta es que TMS es muy intuitivo, siga estos pasos.

Se debe conectar al sitio o página donde "apunta" el TMS que ha adquirido, donde le aparecerá una pantalla similar a esta.

Debe de indicar los datos del idioma, usuario y contraseña para poder ingresar al sistema Task Management System.

Idioma:  Indique el idioma con el que desea se muestre el menú de opciones del TMS.  Por default siempre se inicializa en español, pero también puede indicar el idioma Inglés.

Usuario: Nombre del usuario que va a ingresar al sistema.  Recuerde cada usuario tiene diferentes permisos de acceso a las opciones de menú.

Contraseña: Digite la contraseña o clave de paso asociada al usuario.

Si Usted ha ingresado al sistema como usuario ADMIN, lo primero que va poder visualizar es un resumen de todos los proyectos que están en curso como se muestra en la siguiente pantalla.

Una vez conectado, el usuario puede utilizar los botones de cada opción. Dentro de cada opción del  TMS, usted puede utilizar botones para agregar, eliminar, modificar y editar, son los mismos para todas las opciones.

Inicio

Es la pantalla principal muestra todos los proyectos que se encuentran en la base de datos en el momento de ingresar al sistema

Proyectos

Pantalla que contiene los nombres de los proyectos, sus dueños, fecha de inicio o final del mismo, las tareas asignadas a cada uno de ellos, además se lleva el control de las tareas asignadas, atrasadas, sin asignar y los tickets.

Agenda

Se utiliza para programar la agenda electrónica.

Clientes

En esta sección se muestran la organización de los clientes.

Reportes

En esta sección muestra los reportes

Buscar

Esta sección se utiliza para realizar criterios de búsqueda de acuerdo a los requerimientos del usuario.  

Admin

A este menú solamente tienen acceso los administradores de la base de datos y se ingresan, modifican, eliminan todos lo relacionado a la parte administrativa.

Esta pantalla es utiliza para consultar los datos del usuario que se encuentra trabajando con el CRM/SOS. Estos datos pueden ser modificados, también puede eliminar un usuario de la base de datos, pero solamente por el administrador del sistema.
