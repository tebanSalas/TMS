Descripción de Íconos

Usted ve los iconos de ésta manera cuando no se encuentra posicionado en una línea de detalle determinada.

 Cuando Usted se posiciona en los check de selección para trabajar con un proyecto, tarea, ticket, etc los iconos cambian de color.

 

VER:  Despliega toda la información asociada a la pantalla que se está consultando. Con este icono el usuario solamente puede consultar, si lo que desea es modificar la información contenida debe utilizar el ícono EDITAR.

AGREGAR:  Este ícono es para agregar un nuevo registro a la base de datos.

ELIMINAR:  Elimina un registro de la base de datos.

EDITAR:  Realiza una consulta a la base de datos, para recuperar todos los registros de la entidad asociada al formato y luego poder modificarlos.

COPIAR:  Permite generar una copia de los datos que existen en el sistema.

AGREGAR SITIO: Se utiliza para que aparezcan los datos en el portal del cliente.  Los datos que pueden aparecer son: Tareas, Proyectos, Tickets, Miembros del Equipo de trabajo para cada uno de los proyectos.

QUITAR DEL SITIO DEL PROYECTO: Se eliminan datos del portal del cliente.

TRASLADAR: Ese botón se utilizar para trasladar un tarea determinada a otro miembro definido en el equipo de trabajo del proyecto.

CERRAR: Se utilizar para cerrar un ticket, o sea ninguna otra persona puede agregar más discusiones al mismo, se da por terminado. Este botón solamente se encuentra en la opción Tickets.

 Significa que el proyecto o tarea se finalizó.

MOSTRAR TODO:  Realiza una consulta a la base de datos, para recuperar todos los registros de la entidad asociada al formato sin criterio de búsqueda particular.
