AGREGAR UN SEGUIMIENTO ADMINISTRATIVO

Esta pantalla se utiliza para incluir un nuevo proceso administrativo que necesite la oportunidad o grupo de oportunidades.

Nombre Nombre del nuevo proceso administrativo.

Descripción: Describa detalladamente en que consiste el proceso administrativo

Estado: Estado en que se encuentra el proceso administrativo. Puede tener los siguientes valores:

Iniciado: El proceso ya se inició

No iniciado: Se incluyó la gestión administrativa pero aún no se inicia el trámite.

Finalizado: El proceso administrativo ya finalizó.

Fecha inicio: Fecha en que debe iniciar la gestión administrativa.

Fecha final: Fecha en que se estima finaliza la gestión administraría Utilice el siguiente botón para desplegar la ventana de la agenda .

Comentarios: Campo para anotar alguna observación o comentarios que se estime importante tomar en cuenta para la gestión administrativa.

Funciones de contactos

Para cada cuenta debe asignarse las personas contactos. En esta pantalla se ingresan las personas (contactos) con los que se deben tratar para negociar.

SOCIOS

Los socios son las personas que ayudan a entrar a una cuenta. Estos socios ganan un porcentaje de comisión.

Socios: Seleccione el socio que ayuda a entrar en la cuenta.  Estos socios deben esta definidos por el Administrador del sistema.

Porcentaje: Porcentaje de comisión que se le da al socio, en algunos casos el socio no tiene comisión.

AGREGAR COMPETIDORES

A los competidores se les define una cuenta también, y se definen para establecer cuales son las áreas fuertes de mi competencia.

Competidor: Seleccione el tipo de competidor de la lista. Debe estar previamente establecidas en las Cuentas.

Fortalezas: Describa claramente cuáles son las fortalezas de ese competidor para la oportunidad de negocio.

Debilidades: Describa claramente cuáles son las debilidades de mi compañía para la oportunidad de negocio.

Plan de Acción: Indique detalladamente el plan de acción que se tiene que hacer para poder superar las debilidades que tenemos.

AGREGAR CONTENIDO ENLAZADO

Esta opción es muy importante, todas las ofertas que se le hacen al cliente deben estar incluidas en el sistema. Esto nos permite tener acceso directo a la base de datos CRM/SOS  y poder desde cualquier internet y poder visualizar la última oferta que se le hizo al cliente. En los casos en que un vendedor asignado a una cuenta no se encuentra en la empresa y el cliente necesita que lo atiendan, entonces otro vendedor pueda atender al cliente y pueda contar con toda la información necesaria. Los archivos que se suben deben de contener los acuerdos tomados con el cliente,. las hojas de excel por ejemplo: negociaciones de descuentos, etc.

Subir  Sube el documento que tengo en el disco duro y guardarlo en el repositorio del CRM/SOS Ejemplo: elaboro una oferta en Microsoft Word para posteriormente agregarlo al sistema como contenido enlazado ya que de esa forma queda guardado en el sistema.  

Examinar: Utilice el botón Examinar para seleccionar la ruta de acceso a los archivos.

Comentarios:  Campo para anotar alguna observación o comentario que se estime conveniente pera el contenido enlazado.
