Exportando Contactos

Esta opción se utiliza para generar en excel todos los datos que se encuentran en la base de datos de las cuentas, contactos, oportunidades, las cuales se presentan distribuidas de la siguiente manera: Organizador, Dirección, Ciudad, Web Site, Apartado, Teñéfono, E-mail, Tipo, Territorio, Sector, Título, Título personal, Ocupación, Nombre, Celular, Teléfono del trabajo, Teléfono casa, E-mail principal, E-mail Secundario, Fax, Beeper, Dirección, Cumpleaños, Notas.

El sistema siempre le hace la siguiene pregunta:

Abrir: Abre los datos en Excel.

Guardar: Guarda toda la información en un archivo. El sistema le pregunta la ruta donde desea se guarde la información.

Cancela: No ejecuta la generación de datos.

Más información: Abre la ayuda en línea.
