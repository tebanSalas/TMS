Instalando mi TMS por primera vez

TMS viene preinstalado de fabrica en un equipo, de tal forma que lo único que tiene que hacer es lo siguiente.

  1. Sacar el equipo de su caja

  2. Conectar el monitor

  3. Encender la máquina

  4. Conectar el equipo a su red interna

  5. Configurar el puerto TCP-IP del equipo

  6. Ir a la sección de configuración o parametrización del sistema

Ediciones de TMS

TMS puede ser usado por empresas de todo tamaño, todo depende del número de miembros y administradores que lo utilicen.

La edición para empresas pequeñas es de 5 usuarios de TMS, fué desarrollada para empresas PYMES (Pequeña y mediana empresas)  es una solución de administración de clientes práctica y rentable para aquellos equipos que deseen centralizar y compartir de forma rápida, sencilla y económica las actividades y la información del cliente para aumentar la productividad inmediatamente.

  • Viene configura para 5 usuarios

  • Trae un servidor de un procesador

  • Trae Oracle Base de datos Standard Edition One

  • Linux Red Hut preinstalado

  • Administración de clientes en colaboración

  • Actividades, tareas y eventos en colaboración

  • Informes

  • Servicio de atención al cliente básico por teléfono, Web o correo electrónico gratuito para el primer año.

Para más de 5 usuarios, se requiere que active por internet las licencias adicionales, esto incluye la base de datos Oracle y la aplicación para un usuario adicional.

1. El producto se accesa en forma directa desde cualquier navegador de internet.

2. No requiere ninguna instalación previa

3. Es recomendable que si usa Microsoft Explorer, modifique estos parámetros.

En opciones (Options) del navegador, en la parte Temporary Internet Files, seleccione la sección Settings y marque la opción Every visit to the page, sino el sistema le podría presentar errores de refrescamiento de páginas.
