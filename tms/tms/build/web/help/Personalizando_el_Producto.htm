Personalizando el Producto

TMS le permite a usted configurar una serie de opciones que son muy importantes para que inicie con éxito el uso de este poderoso software.

Lo primero que debe conocer es que la única persona que puede configurar el sistema es el usuario que tiene perfil de administrador, es decir, el tiene un nombre y password especial, el cual le dará acceso a un menú de opciones ampliado que no tienen los demás usuarios.

Le debe salir una pantalla como esta, observe que tiene un botón que se denomina ADMIN.

Para dar inicio, usted debe configurar el producto, debe usar la opción de Configuración del sistema.
